Welcome to the Zerp user manual. Zerp is an all-in-one business ERP where a Super Admin manages multiple companies, and each company runs projects, accounting, HRM, CRM, and POS from its own secure workspace. This guide is built for anyone running Zerp. Inside, you’ll find step-by-step instructions covering login, dashboard, company and plan setup, projects, accounting, HRM, CRM, POS, and system settings.
